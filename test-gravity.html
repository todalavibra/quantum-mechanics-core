Physics Parameters

You can customize the gravity hack with these options:

  • Quantum Probability: 0-100% (controls randomness/jitter)
  • Antigravity: true/false (invert gravity direction)
  • Time Dilation: 0.5-5.0 (slow motion effect)
  • Dispersal: true/false (explosive radial force)

Example Code

// Activate with custom settings
activateGravity({
    quantumProbability: 50,
    antigravity: false,
    timeDilation: 1.0,
    dispersal: true
});

// Deactivate
deactivateGravity();
